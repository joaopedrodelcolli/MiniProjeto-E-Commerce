# Mini Projeto – Análise de Dados E-commerce

 **Sobre o projeto**

Este projeto foi desenvolvido com o objetivo de praticar análise exploratória de dados utilizando Python.
A base contém informações de um e-commerce, incluindo pedidos, itens vendidos e formas de pagamento.

O foco principal foi integrar os datasets, realizar o tratamento dos dados e gerar métricas de negócio para análise de desempenho.

---

**Objetivos:**

* Praticar manipulação de dados com Pandas

* Realizar limpeza e conversão de tipos

* Integrar múltiplas tabelas

* Calcular métricas de negócio

* Criar visualizações para análise de faturamento

---

**Tecnologias utilizadas:**

* Python

* Pandas

* NumPy

* Matplotlib

* Seaborn

* Google Colab

---

**Etapas da análise:**

1. Entendimento e exploração inicial dos dados

2. Tratamento e padronização das colunas

3. Integração das tabelas por meio de chaves em comum

4. Cálculo de métricas como:

    * Faturamento total

    * Ticket médio

    * Volume de pedidos

5. Análise temporal do faturamento

6. Visualização dos principais indicadores

---

**Principais análises:**

* Evolução do faturamento mensal

* Identificação dos meses com maior receita

* Distribuição das formas de pagamento

* Cálculo de indicadores gerais do período

---

**Objetivo do projeto**

Este projeto faz parte do meu processo de aprendizado em Análise de Dados e tem como finalidade consolidar conhecimentos em manipulação, transformação e exploração de dados com Python.
